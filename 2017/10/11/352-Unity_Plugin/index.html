语法说明

  • path 是菜单路径；
    • 一级菜单名称不支持中文
  • $t 是一个快捷键实例，在路径后面接空格，然后加上快捷键表示，单纯的一个按键快捷键按键字符前带下划线。该项非必需
    • % 表示 ctrl
    • # 表示 shift
    • & 表示 alt
  • Is 设置为true的时候，如果没有选中游戏对象，会显示不可用状态，该选项非必需
  • Priority 是优先级，数值越小优先级越高，非必需，其默认值为1000。
